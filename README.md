## 📖 About This Repository

This repository contains all the smart contracts I developed during my **Blockchain Development Internship** at CodeAlpha. Each file represents a different project that helped me learn Solidity and blockchain development.

---

## 🗂️ Projects I've Built

### 1️⃣ Simple Storage Smart Contract
**File:** `MyNumberStorage.sol`

This was my first smart contract! I built a simple digital counter that can:
- ✅ Store a number on the blockchain
- ✅ Increase the number by 1 (increment)
- ✅ Decrease the number by 1 (decrement) 
- ✅ Read the current value anytime

---

### 2️⃣ Multi-Send Smart Contract
**File:** `MultiSend.sol`

I built a contract that can send Ether to multiple people at once:
- ✅ Accept an array of Ethereum addresses
- ✅ Send equal amounts to everyone in one transaction
- ✅ Loop through all receivers automatically
- ✅ Handle Ether distribution safely

---

### 3️⃣ Polling System Smart Contract
**File:** `PollingSystem.sol`

I created a decentralized voting system:
- ✅ Anyone can create a poll with options
- ✅ People can vote before the deadline
- ✅ One person can vote only once (no cheating!)
- ✅ Winner is announced automatically after voting ends

---

### 4️⃣ Crypto Locking Smart Contract
**File:** `CryptoLocking.sol` (Bonus Task)

This was like creating a Fixed Deposit (FD) system on blockchain:
- ✅ Users can deposit Ether with a lock time
- ✅ Withdrawals are blocked before the lock time ends
- ✅ After the lock time, anyone can withdraw their funds
- ✅ Secure time-lock mechanism using block.timestamp

## 🚀 How to Run These Projects

1. **Open Remix IDE:**

 https://remix.ethereum.org
 
2. **Create a new file** with the `.sol` extension

3. **Copy the code** from any file in this repository

4. **Compile** the code (green tick should appear)

5. **Deploy** the contract on Remix VM (London)

6. **Test** all functions to make sure they work
